# 动画信息区域布局调整说明

## 修改日期
2026-01-19

## 修改目标
将动画广场的动画信息区域（包括动画名称、点赞数、作者、分享按钮）从画布下方移动到动画输入框的右下角外侧，与"已选择"和"约束关系"这些信息齐平。

## 修改内容

### 1. PhysicsPage.jsx
**修改位置：** 第 68-81 行

**修改内容：**
- 将 `PhysicsInputBox` 和 `AnimationInfoBar` 包裹在一个 `position: relative` 的 div 中
- `AnimationInfoBar` 作为独立组件显示在输入框外侧
- 移除了之前传递给 `PhysicsInputBox` 的 `plazaAnimationInfo` 和 `onClosePlazaInfo` props

**布局结构：**
```jsx
<div style={{ position: 'relative' }}>
  <PhysicsInputBox />
  {plazaAnimationInfo && (
    <AnimationInfoBar 
      animationInfo={plazaAnimationInfo}
      onClose={() => setPlazaAnimationInfo(null)}
    />
  )}
</div>
```

### 2. AnimationInfoBar.jsx
**修改位置：** 第 25-36 行

**修改内容：**
- 将容器定位方式从 `margin: '12px 0'` 改为 `position: 'absolute'`
- 设置 `marginTop: 8` 使其紧贴在输入框下方
- 保持 `maxWidth: 'calc(100vw - 450px)'` 与输入框宽度一致
- 添加 `zIndex: 10` 确保不被其他元素遮挡

**样式调整：**
- 标题字体从 16px 调整为 14px
- 点赞按钮尺寸从 medium 改为 small
- 作者信息字体从 13px 调整为 12px
- 分享按钮 padding 从 `6px 12px` 调整为 `4px 10px`，字体从 13px 调整为 12px
- 关闭按钮尺寸从 24x24 调整为 20x20
- 描述文字字体从 13px 调整为 12px
- padding 从 16px 调整为 12px

### 3. PhysicsInputBox.jsx
**清理内容：**
- 移除了 `LikeButton` 和 `ShareLinkModal` 的导入（不再需要）
- 移除了 `plazaAnimationInfo` 和 `onClosePlazaInfo` props
- 移除了 `showShareModal` 状态
- 移除了内嵌的广场动画信息显示区域（约 120 行代码）
- 移除了分享链接弹窗组件

## 最终效果

### 布局层级
```
PhysicsPage
└── div (position: relative)
    ├── PhysicsInputBox
    │   ├── 上传区域
    │   ├── 识别到的元素
    │   ├── 已选择
    │   └── 约束关系
    └── AnimationInfoBar (position: absolute, 紧贴在输入框下方)
        ├── 标题 + 点赞 + 作者 + 分享按钮
        └── 描述
```

### 视觉效果
- 动画信息区域显示在输入框的正下方（外侧）
- 与"已选择"和"约束关系"保持相同的左对齐
- 宽度与输入框一致
- 使用更紧凑的样式（更小的字体和间距）
- 保持所有交互功能（点赞、分享、关闭）

## 代码优化
- 删除了 PhysicsInputBox 中约 150 行的冗余代码
- 保持了单一职责原则：AnimationInfoBar 专门负责显示动画信息
- 简化了组件间的 props 传递

## 兼容性
- 保持了原有的所有功能
- 不影响"我的动画"面板和"动画广场"面板
- 响应式布局保持不变
